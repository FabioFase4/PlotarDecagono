# Projeto
- Plotagem de um Decágono na Tela a partir de Certa Cor de Síntese Escolhida pelo Usuário
- Projeto desenvolvido para a disciplina de Programação para Dispositivos Móveis no curso de Informática da **UNIVAP**.
- Orientado por: Wagner dos Santos Clementino de Jesus

# Recursos Usados
- Visual Studio (IDE)
- Windows Forms (.NET Framework)
- C# (Linguagem)

# Técnicas Adotadas
- Orientação a Objetos (Indispensável)
- Economia de Memória
- Uso de Primitivas Gráficas
- Métodos para Plotagem Gráfica
- Limpeza de Código
- Técnicas Primordiais de Orientação a Objetos
